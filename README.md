## Supplier Weather ETL Project
This project extracts supplier transactions, transforms them with SQL queries, 
and integrates NOAA weather data using Python and Snowflake.

**Tech Stack:** Python 路 SQL 路 Snowflake 路 Pandas 路 XML Parsing  
**Key Features:**
- Extracts and transforms supplier data from XML
- Loads cleaned data into Snowflake tables
- Joins weather metrics with supplier locations
- Demonstrates data integration and analytics pipeline skills
